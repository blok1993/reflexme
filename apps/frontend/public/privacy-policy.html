Политика конфиденциальности

Последнее обновление: 23 апреля 2025 г.

Настоящая Политика конфиденциальности описывает, какие данные собирает приложение ReflexMe, как они используются и какие права есть у пользователя. Используя приложение, вы соглашаетесь с условиями настоящей политики.

1. Кто мы

ReflexMe — приложение для ежедневной саморефлексии и персональных прогнозов. Оператор: reflexme.ru. По вопросам конфиденциальности пишите на hello@reflexme.ru.

2. Какие данные мы собираем

ReflexMe собирает минимально необходимые данные для работы приложения:

  • Анонимный идентификатор устройства — случайный UUID, генерируется при первом запуске и хранится локально. Не привязан к личности пользователя.
  • Данные чек-ина — настроение, выбранный фокус, короткий текстовый контекст (не обязательно).
  • Оценки прогнозов — «Да», «Частично», «Мимо» по каждому прогнозу вечером.
  • Предпочтения — имя (по желанию), стиль общения, дата рождения (по желанию).

Мы не собираем: имя, email, телефон, геолокацию, контакты, фото или другие персональные данные, позволяющие идентифицировать пользователя.

3. Как мы используем данные

  • Генерация персональных прогнозов дня с помощью ИИ.
  • Формирование еженедельных наблюдений и профиля пользователя для повышения точности прогнозов.
  • Отображение истории и паттернов внутри приложения.

Данные не продаются третьим лицам и не используются для таргетированной рекламы.

4. Передача данных третьим сторонам

Для генерации прогнозов используется API OpenAI (ChatGPT). Данные чек-ина передаются в обезличенном виде в рамках запроса к модели. OpenAI не хранит данные запросов для обучения моделей при использовании API. Подробнее: openai.com/policies/privacy-policy.

Иные передачи данных третьим сторонам не осуществляются.

5. Хранение данных

Данные хранятся на защищённом сервере в облачной инфраструктуре Railway. Идентификатор сессии хранится в браузере пользователя (localStorage). Данные привязаны к устройству — смена браузера или устройства создаёт новую анонимную сессию.

6. Права пользователя

Вы вправе:

  • Запросить удаление всех данных, связанных с вашим анонимным идентификатором.
  • Удалить локальные данные, очистив хранилище браузера (localStorage).

Для удаления серверных данных напишите на hello@reflexme.ru с темой «Удаление данных», указав ваш анонимный ID (можно найти в настройках приложения).

7. Дети

Приложение не предназначено для лиц младше 16 лет. Мы не собираем данные детей намеренно. Если вам стало известно о предоставлении данных ребёнком, свяжитесь с нами.

8. Изменения политики

Мы можем обновлять эту политику. При существенных изменениях дата в заголовке будет обновлена. Продолжение использования приложения после изменений означает согласие с новой редакцией.
